# 🚦 Semáforo Inteligente con Botón de Emergencia

Sistema basado en ESP8266 que simula un semáforo inteligente con control automatizado y un botón de emergencia para situaciones prioritarias.

---

## 🧠 Descripción

Este proyecto implementa un semáforo controlado por un microcontrolador ESP8266, capaz de gestionar el flujo vehicular y peatonal. Incluye un botón de emergencia que permite alterar el ciclo normal del semáforo para dar prioridad en situaciones críticas.

Está diseñado como una solución de automatización vial básica, con enfoque en sistemas embebidos e IoT.

---

## ⚙️ Tecnologías utilizadas

- ESP8266
- Arduino IDE
- Programación en C/C++
- Electrónica básica
- Redes (WiFi integrado en ESP8266)

---

## 🔌 Componentes

- ESP8266
- LEDs (rojo, amarillo, verde)
- Resistencias
- Botón de emergencia
- Cables de conexión
- Protoboard

---

## 🚀 Funcionalidades

- 🚦 Control automático de luces (rojo, amarillo y verde)
- ⏱️ Secuencia temporizada del semáforo
- 🚨 Activación de modo emergencia mediante botón
- 🔁 Reinicio del ciclo después del evento de emergencia

---

## 🛠️ Instalación y uso

1. Abrir el código en Arduino IDE  
2. Seleccionar la placa ESP8266  
3. Conectar el dispositivo al computador  
4. Cargar el código al microcontrolador  
5. Conectar los LEDs y el botón según el circuito
